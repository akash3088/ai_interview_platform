# AI Interview Practice Platform

An AI-powered mock interview system that allows users to practice interviews through voice/text and receive intelligent feedback.

## Features
- AI-generated interview questions
- Voice and text-based answering
- Answer evaluation using AI
- Emotion detection system
- Personalized feedback and improvement plan

## How it Works
1. User selects interview role
2. System asks AI-generated questions
3. User responds via voice/text
4. AI evaluates answers
5. Feedback and improvement plan generated

## Tech Stack
- Python (Flask)
- Gemini API
- MySQL
- HTML/CSS/JS

- Real-time emotion tracking
